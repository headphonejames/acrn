This is my attempt at implementing the Acoustic Coordinated Reset Neuromodulation tinnitus treatment protocol using this paper as a guide. This page will probably only work with a relatively modern html browser.

First try to match your tinnitus frequency using the "Play Single Frequency" button and then start the protocol with the "Play ACRN" button. Individual frequency values and volume can be adjusted by clicking on them.

Inspired by this thread on tinnitustalk.com and this reddit thread. Feedback and suggestions are most welcome.
